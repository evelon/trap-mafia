## 공통 가정사항

### SSE 재연결 정책

SSE 연결이 끊기면 자동 재연결을 시도한다. 일정 횟수/시간 후에도 재연결에 실패하면 유저에게 오류 상태를 알리고, 새로고침 또는 로비 이동 등 수동 행동 수단을 제공한다.

## MVP 공통 가정사항

- **방이 1개로 고정**: 방 생성/목록 조회 없음. 방 ID는 하드코딩.
- **모든 참가자가 호스트**: 방의 모든 참가자가 호스트 권한을 가진다. 게임 시작, 강제 퇴장 등 호스트 액션을 누구나 실행할 수 있다.
- **방 해산 없음**: 마지막 유저가 퇴장해도 방은 사라지지 않는다.
