# 와이어프레임: 방 화면 (Room Screen)

## 화면 1: 방 대기 화면

화면 전체는 스크롤되지 않으며, 모든 요소가 한 화면 안에 고정 배치된다.

### 레이아웃

```text
+------------------------------------------+
| [< 나가기]    방 이름    [설정 요약 ⓘ]     |
+------------------------------------------+
|                                          |
|  참가자 (3/8)                             |
|  +------+ +------+ +------+ +------+     |
|  | (ㅇ) | | (ㅇ) | | (ㅇ) | |      |     |
|  | 유저1 | | 유저2 | | 유저3 | | 빈칸 |     |
|  +------+ +------+ +------+ +------+     |
|  +------+ +------+ +------+ +------+     |
|  |      | |      | |      | |      |     |
|  | 빈칸 | | 빈칸 | | 빈칸 | | 빈칸 |     |
|  +------+ +------+ +------+ +------+     |
|                                          |
+------------------------------------------+
| [           게임 시작           ]  <- 고정  |
+------------------------------------------+
```

### 헤더 영역

- 좌측: "나가기" 버튼 [FR-007]
- 중앙: 방 이름 [FR-001]
- 우측: 방 설정 요약 + ⓘ 아이콘 [FR-003] — 탭하면 설정 상세 모달 열림

### 참가자 그리드

- 8칸 고정 그리드 (4열 × 2행) [FR-002]
- 참가자가 있는 칸: 아바타 + 닉네임 카드
- 빈 칸: 비어있는 상태의 카드로 표시 (아직 참가하지 않은 자리임을 나타냄)
- 다른 유저 입장·퇴장 시 즉시 갱신 [FR-004]

### 하단 고정 버튼

- "게임 시작" 버튼 — 화면 하단 고정, 호스트에게만 표시 [FR-005]
- 탭하면 게임이 시작되고 모든 참가자가 게임 화면으로 이동
- [FR-006] 게임이 이미 진행 중이면 이 화면을 거치지 않고 게임 화면으로 이동

## 모달: 방 설정 상세

헤더의 설정 아이콘을 탭하면 모달로 표시한다.

```text
+---------- 방 설정 ----------+
|                             |
|  최대 인원      8명          |
|  토론 시간      120초        |
|  투표 시간      30초         |
|  ...기타 RoomSettings 항목   |
|                             |
|          [ 닫기 ]           |
+-----------------------------+
```

- 읽기 전용 (설정 변경은 MVP 범위 외)
- RoomSettings 전체 항목 표시 [FR-003]

## 상태: 연결 오류

```text
+------------------------------------------+
| [< 나가기]    방 이름    [설정 요약 ⓘ]     |
+------------------------------------------+
|                                          |
|         연결이 끊어졌습니다.                |
|         재연결 시도 중...                   |
|                                          |
+------------------------------------------+
|                                          |
+------------------------------------------+
```

- [FR-008] 실시간 연결 오류 시 참가자 영역 위에 오류 상태 오버레이 표시
- 자동 재연결 시도 중에는 "재연결 시도 중..." 메시지
- 재연결 실패(일정 횟수/시간 초과) 시:

```text
|         연결에 실패했습니다.                |
|                                          |
|    [ 새로고침 ]    [ 로비로 이동 ]          |
```

## 상태: 게임 시작 실패

- "게임 시작" 요청 실패 시 토스트로 에러 메시지 표시
- 방 화면에 그대로 머무름
